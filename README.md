# 🚚 YUA Delivery - 日本ユーア東海 配送アプリ

株式会社 日本ユーア東海の配送業務を効率化する、シンプルな配送管理アプリです。

## 🌐 アクセスURL

**https://nexcityuniseed-maker.github.io/yua-delivery/**

スマホのブラウザで上記URLを開いてください。ホーム画面に追加すればアプリのように使えます。

## 📱 主な機能

| 機能 | 説明 | 対象 |
|------|------|------|
| 🚗 **今日の配送ルート** | 配送先の案内・完了チェック・地図ナビ | ドライバー |
| 📥 **リスト取込** | CSV・PDFから配送リストを読み込み | 管理者 |
| 🗺 **ルート作成** | 車両別にルートを自動作成 | 管理者 |
| 🏪 **配送先マスタ** | 店舗情報の追加・編集 | 管理者 |

## 📁 ファイル構成

```
yua-delivery/
├── index.html    ← トップページ（メニュー）
├── driver.html   ← ドライバー用配送画面
├── import.html   ← リスト取込画面
├── route.html    ← ルート作成画面
└── shops.html    ← 配送先マスタ管理画面
```
